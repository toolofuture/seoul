## 2. 핵심 분석 방법론 (Critical Methodology)

### 🦋 생존율 분석 (Butterfly Chart)
> *"많이 생기는 것보다 중요한 건, 얼마나 살아남느냐입니다."*
- **문제 의식:** 신규 개점이 많아도 폐점이 그만큼 많다면 그것은 유망 업종이 아니라 '전쟁터'입니다.
- **해결:** 신규 개점(Blue)과 계약 종료/해지(Red)를 대조하는 **버터플라이 차트**를 도입하여 시장의 과열도를 시각화했습니다.

### 🌫️ 정보 비대칭 시각화 (Missing Data Viz)
> *"보이지 않는 데이터가 가장 위험합니다."*
- **문제 의식:** 매출액을 공개하지 않는 브랜드는 투자 리스크가 매우 높습니다.
- **해결:** 매출 데이터가 '0'이거나 미기재된 브랜드는 차트에서 **흐릿하게(Blur)** 처리하거나 ⚠️ 경고 배지를 부착하여 직관적인 위험 신호를 보냈습니다.

### 💎 숨겨진 알짜 발굴 (Hidden Gems)
> *"규모보다 효율입니다."*
- **문제 의식:** 매장 수가 적다고 무시해서는 안 됩니다.
- **해결:** **산점도(Scatter Plot)**를 통해 점포 수는 적지만(초기 진입), 면적당 매출 효율이 압도적인 '강소 브랜드'를 발굴했습니다.

## 3. 주요 인사이트 (Key Findings)

1.  **🚨 '순증가 1위'의 착시:**
    *   외식업 1위 **골프존파크(+42개)**는 소자본 창업이 아닌 고비용 장치 산업입니다. 이는 대중성보다는 **'자본의 쏠림'** 현상으로 해석해야 합니다.
2.  **🎓 노동 집약적 성장의 한계:**
    *   **학원/교습소(아트앤하트 등)**의 폭발적 성장은 낮은 진입 장벽에 기인합니다. 낮은 평균 매출은 1인 운영의 한계를 보여주므로, 무리한 확장보다는 **운영 효율화(AI/자동화)**가 필수적입니다.
3.  **📉 구조적 하락의 시그널:**
    *   **토즈(-23%), 팔공티(-46%)** 등의 감소는 개별 브랜드의 실패가 아닌, 스터디카페 무인화 및 저가 시장 포화라는 **산업 구조의 변화(Digital Shift)**를 반영합니다.
